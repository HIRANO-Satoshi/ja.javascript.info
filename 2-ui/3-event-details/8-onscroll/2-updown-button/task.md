# Up/down button

ページスクロールを助けるための "トップへ" ボタンを作成してください。

次のように動作します:
- ページが少なくともウィンドウの高さよりスクロールされていない間は見えません。
- ページがウィンドウの高さよりもスクロールされたとき、-- 左上端に "上向き" の矢印が表示されます。ページがスクロールで上に戻されると消えます。
- 矢印をクリックすると、ページはトップへスクロールします。
